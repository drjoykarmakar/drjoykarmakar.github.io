# Upload Instructions

## Option A — Upgrade the GitHub Pages website

Use this for: `https://github.com/DrJoyKarmakar/DrJoyKarmakar.github.io`

1. Download and unzip `joy-world-class-profile.zip`.
2. Open your `DrJoyKarmakar.github.io` repository on GitHub.
3. Upload these items to the repository root:
   - `index.html`
   - `assets/`
   - `README.md`
   - `REPO_README_TEMPLATE.md`
   - `UPLOAD_INSTRUCTIONS.md`
4. Commit changes.
5. Go to **Settings → Pages** and confirm GitHub Pages is publishing from the correct branch/root.
6. Visit `https://drjoykarmakar.github.io` after deployment.

## Option B — Upgrade your GitHub profile README

Use this for the special profile repository:

```text
https://github.com/DrJoyKarmakar/DrJoyKarmakar
```

1. Create the repository if it does not exist. It must be named exactly `DrJoyKarmakar`.
2. Copy `PROFILE_README.md`.
3. Rename it to `README.md`.
4. Upload it to the root of `DrJoyKarmakar/DrJoyKarmakar`.
5. Commit changes.
6. Visit `https://github.com/DrJoyKarmakar`.

Keep the profile README mission-focused and the GitHub Pages site portfolio-focused. Use each repository README as a paper-quality documentation page.
