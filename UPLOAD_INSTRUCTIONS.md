# Upload Instructions

## A. GitHub profile README

Your GitHub profile README lives in this repository:

```text
DrJoyKarmakar/DrJoyKarmakar
```

Steps:

1. Open `README.md` from this folder.
2. Copy its full content.
3. Paste it into `DrJoyKarmakar/DrJoyKarmakar/README.md`.
4. Upload the `assets/svg/hero.svg` file to the same repository path:

```text
assets/svg/hero.svg
```

5. Commit changes.

Important: the profile README only appears on your GitHub profile if the repository name exactly matches your username.

---

## B. GitHub Pages website

Your public website repository is:

```text
DrJoyKarmakar/DrJoyKarmakar.github.io
```

Steps:

1. Upload these files/folders to the root of `DrJoyKarmakar.github.io`:

```text
index.html
assets/
projects/
```

2. Commit changes.
3. Go to `Settings -> Pages`.
4. Set source to `Deploy from a branch`.
5. Select `main` and `/root`.
6. Visit:

```text
https://drjoykarmakar.github.io
```
