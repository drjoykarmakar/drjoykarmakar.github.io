## Mission

I work at the interface of **synthetic medicinal chemistry**, **chemical biology**, and **machine learning** to build open scientific infrastructure for discovering, evaluating, and validating molecules.

My current focus is a research ecosystem for **AI-driven molecular discovery**:

```text
open data standards -> benchmark datasets -> ML baselines -> active learning -> prospective validation
```

Rather than optimizing molecules in isolation, my work treats the **experiment** as the atomic unit: analyte, recognition mechanism, sensor molecule, assay protocol, signal, selectivity, uncertainty, and the next experiment.
